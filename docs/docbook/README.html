mybastisnet - getting and setting up docbook utilities

Docbook is a is an XML documentation format specification, essentially implemented in two things that a tool needs to parse a docbook source document (in XML):

  1. an XML DTD (docbookx.dtd). This is available in many, many versions. From the headers in the mybatisnet source docbook documents, I deduced version 4.1.2 is the one the source files are expecting. You will find the docbook DTD and its dependencies in the .\dtd folder. The original source for the DTD is http://www.oasis-open.org/docbook/xml/4.1.2/.
  2. a set of XML stylesheets (xsl files), which are provided at http://sourceforge.net/projects/docbook. These instruction were tested with the latest stable release, which as of this writing is docbook-xsl-ns v1.76.0.
    1. Download the stylesheets bundle at http://sourceforge.net/projects/docbook/files/docbook-xsl-ns/1.76.0/docbook-xsl-ns-1.76.0.zip/download.
    2. extract the contents of the top-level folder from the ZIP archive in the .\styles folder. You should wind up with the following folders under docs/docbook/styles: common, docsrc, eclipse, ... As for the DTD, these file locations are currently hardcoded in the mybatisnet docbook stylesheets, so it's important to unzip the files in the right place.
