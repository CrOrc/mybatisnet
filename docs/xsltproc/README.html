mybastisnet - getting and setting up xsltproc and related utilities

xsltproc is one of many tools that are able to apply stylesheets to XML documents, with the usual goal of converting the XML to some other format. To get a windows version of xlstproc:

  1. Download the following files:
    1. ftp://ftp.zlatkovic.com/libxml/oldreleases/iconv-1.9.1.win32.zip
    2. ftp://ftp.zlatkovic.com/libxml/oldreleases/libxml2-2.6.15.win32.zip
    3. ftp://ftp.zlatkovic.com/libxml/oldreleases/libxmlsec-1.2.6.win32.zip
    4. ftp://ftp.zlatkovic.com/libxml/oldreleases/libxslt-1.1.12.win32.zip
    5. ftp://ftp.zlatkovic.com/libxml/oldreleases/zlib-1.2.1.win32.zip
  2. Unzip each of the files above in this folder (e.g. docs/xsltproc).
  3. Copy the contents of all the bin folders (EXEs and DLLs) in this folder (e.g. docs/xsltproc).
  4. Add this folder to your path, so that xsltproc.exe is available from a command prompt.
